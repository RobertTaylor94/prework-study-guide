HTML

  • The head element contains information about the webpage
  • The body element represents the visible information show to a user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Height and Width specify the size of the element.
  • Color indictates the colour of an element and can be named or chosen using its hex code.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the main branch code from the cloud
  • git add -A: commit all files in the working branch to the staging area
  • git commit -m: commit files to the repository (-m is used to add a message)

JavaScript

  • Use Cmd+Opt+I to open Chrome Dev Tools
  • console.log() prints the contents of the brackets to the console
