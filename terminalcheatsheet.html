Terminal Commands Cheat Sheet

Navigation

  • cd [path of directory] : Navigates to the chosen directory
  • cd ~ : Navigates to the home directory
  • cd .. : Moves one directory up
  • ls : Lists the files and folders in the current directory
  • mkdir [name of new folder]: creates a new folder in the current directory
  • touch [name of new file]: creates a new file in the current directory
  • rm [name of file] : deletes the named file
  • rm -r [name of folder] : deletes the named folder
  • open [name of file] : opens the named file (MacOS)

Git Commands

  • git status : Returns the current repository
  • git pull origin main : pulls any updates from the main branch down to the current working branch
  • git add -A : Add all changes to the staging area
  • git commit -m "commit messsage" : Commit changes to the working branch with a commit message
  • git push origin _branch name_ : push local changes to the remote branch
